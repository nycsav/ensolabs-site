# Enso Labs — ensolabs.ai

AI transformation consultancy, agentic systems studio, and financial AI product lab.

## Tech Stack
- **Framework:** Next.js 14 (App Router, TypeScript)
- **Styling:** Custom CSS design system (OKLCH color space)
- **Deployment:** Vercel (auto-deploy from GitHub)
- **SEO:** JSON-LD schemas (Organization, Person, ProfessionalService, Product, FAQ, LocalBusiness, Article, Breadcrumb)
- **AEO/GEO:** Definition-lead sentences, MCP endpoint, LLM bot access in robots.txt
- **Content:** MDX-ready, RSS feed, signal2noise integration

## Built With AI
- **Strategy & Research:** Claude Chat (Opus 4.6)
- **Visual Prototyping:** Claude Design
- **Production Code:** Claude Code
- **Content Intelligence:** signal2noise (PlannerAPI)
- **Browser Automation:** Claude in Chrome
- **File Operations:** Desktop Commander
